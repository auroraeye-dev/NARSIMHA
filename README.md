# 🌌 About IMC Prosperity 4

IMC Prosperity is a global **algorithmic trading competition** where participants build trading agents to operate in simulated markets.

It combines:
- 📊 Quantitative modeling  
- 🤖 Algorithmic trading  
- 🧠 Strategic decision-making  
- ⚡ Dynamic market environments  

Participants compete across multiple rounds with increasing complexity, involving both:
- **Algorithmic trading**
- **Manual trading challenges**

---

# 🪐 Theme: Trading on Intara

The competition takes place on **Intara** an exoplanet, where participants establish a **Trade Outpost**.

> 🎯 Goal: Earn **≥ 200,000 XIRECs** profit to qualify for advanced trading phases.

- Overfitting reduces robustness in later rounds  
- Simplicity often outperforms complex strategies
